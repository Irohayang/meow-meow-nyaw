# Installation
## Dependecies
     ncurses
     gcc
     cmake
     
1. install the Dependecies
2. Clone this repo and build the binary
####
     git clone https://github.com/Anik200/nimki/
     cd nimki
     sudo make install
     
# Usage
- save files with [ctrl + s]
- exit the editor with [ctrl + q] or [ctrl + c]
- find text with [ctrl + f] 
- undo with [ctrl + z]
